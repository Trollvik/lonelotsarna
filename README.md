# Lönelotsarna

Hemsida för Lönelotsarna — en ideell förening som verkar för jämställda löner.

## Länkar

- **Sajt:** https://trollvik.github.io/lonelotsarna/
- **English:** https://trollvik.github.io/lonelotsarna/en/
- **Analytics:** https://lonelotsarna.goatcounter.com
- **Releases:** https://github.com/Trollvik/lonelotsarna/releases

## Redigera sajten

Se **[CONTENT-GUIDE.md](CONTENT-GUIDE.md)** för instruktioner om hur du redigerar texter, laddar upp dokument och hanterar nya versioner av Analys Lönelots.

## Utveckling

```sh
npm install     # Installera beroenden
npm run dev     # Starta lokal server (localhost:4321)
npm run build   # Bygge till ./dist/
```

**Teknikstack:** Astro 6, Tailwind CSS 4, GitHub Pages, GoatCounter.
